# haven-monk-mode

*A pace-setting work OS.*  
**Slow enough to notice.**

---

## What this is

Haven Monk Mode is a deliberately slowed working environment designed to protect attention, deepen presence, and restore dignity to focused work.

It does not help you do more.

It helps you **pay attention to what matters**.

---

## The core belief

> Speed is not neutral.  
> Pace shapes perception.  
> Perception shapes meaning.

Most software optimizes for throughput.  
Haven optimizes for **depth entered**.

---

## Monk Mode (the heart of Haven)

Monk Mode is a **pace engine + interface philosophy** that gently slows the user down until attention can land.

It enforces:
- One intention at a time
- Gentle friction before action
- Reflection after completion
- Rest as part of the work loop
- Interfaces without twitch, urgency, or noise

This is not productivity software.  
It is **tempo software**.

---
