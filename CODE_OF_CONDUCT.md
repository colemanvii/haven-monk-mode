# Code of Conduct

Haven Monk Mode is built with care.

We expect:
- Kindness and clarity
- Thoughtful disagreement
- Respect for attention (yours and others')
- No harassment, discrimination, or intimidation

We value:
- Stillness over performative intensity
- Careful language over sarcasm
- Truthful feedback without cruelty

If you experience or witness a violation, open an issue labeled `conduct`
or contact the maintainers via the SECURITY contact method.

Thank you for helping keep this space calm and constructive.
