# Monk Mode Specification

## State Machine

Idle → Orient → Commit → Work → Reflect → Rest → Idle

States cannot be skipped.
Transitions cannot be rushed.

## Orient
Choose one intention.
Plain language only.
No lists, no backlog browsing.

## Commit
Define what “done” means.
Confirm in sentence form.

## Work
Single lane.
Restricted navigation.
Soft time awareness.

## Reflect
One or two questions.
What changed?
What was noticed?

## Rest
Mandatory pause.
No calls to action.
The system waits with you.
